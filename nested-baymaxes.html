Nested Baymaxes

Problem Statement

Suppose function baymax(n) = a string composed of 2^n "o"s, and 2(2n-1) dashes (-) in the arrangement of a baymax (o--o). This means,
baymax(1) = (o--o)
baymax(2) = ((o--o)--(o--o))
baymax(3) = (((o--o)--(o--o))--((o--o)--(o--o)))
baymax(4) = ((((o--o)--(o--o))--((o--o)--(o--o)))--(((o--o)--(o--o))--((o--o)--(o--o))))
and so on.
Given n (n>0), write a function to find baymax(n).
